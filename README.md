# End-to-End Business Performance & KPI Optimization Analysis

## Project Overview
This project analyzes retail sales data to evaluate business performance, identify profit drivers, and uncover loss areas using Excel, SQL, Python, and Power BI.

## Dataset
- Source: Sample Superstore Sales Dataset  
- Provider: Kaggle  
- Link: https://www.kaggle.com/datasets/jr2ngb/superstore-data

The dataset contains retail order-level information including sales, profit, discount, category, region, and time-based attributes.

## Tools & Technologies
- Excel (Data Cleaning, KPIs, Pivot Tables)
- SQL (Trend and Profitability Analysis)
- Python (EDA using Pandas & Matplotlib)
- Power BI (Interactive Dashboard)

## Key Analysis
- Regional and category-level performance evaluation
- Discount impact on profitability
- Monthly sales trend analysis
- Identification of loss-making segments

## Business Insights
- Technology category yields highest profit margins
- Furniture category suffers margin erosion due to heavy discounting
- Western region underperforms in profitability despite high sales

## Outcome
The analysis provides actionable insights to optimize pricing strategies, improve margins, and support data-driven decision-making.
